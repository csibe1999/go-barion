User information structure

This structure represents information about a user in the Barion Smart Gateway process. These users can be either payers or payees related to a payment.

Included in
User information is used in the following structures:

DetailedPaymentTransaction

| Property name |  Property type  |                 Description                  |
| :-----------: | :-------------: | :------------------------------------------: |
|     Name      | NameInformation | Structure representing the name of the user. |
|     Email     |     string      |       The e-mail address of the user.        |
